Add Sale Invoice Tags

This module enables the feature to add tags on invoice form and allow user to pass tags from sales order to invoice.
Supporting for
  * Invoice from Sales order
  * Advance Invoice
  * Percentage bases invoice from Sales Order.

Sale Order Tags

Tags display on Quotation/Sale Order form will be pass to respective invoices of that sales order.

Invoice Tags

Tags from sale order will be shown here. Account users may add more tags.
